Installing

PECL

pecl install raphf

PHARext

Watch out for PECL replicates and pharext packages attached to releases.

Checkout

git clone github.com:m6w6/ext-raphf
cd ext-raphf
/path/to/phpize
./configure --with-php-config=/path/to/php-config
make
sudo make install
